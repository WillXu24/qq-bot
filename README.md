如果你想要安装在自己的服务器：

- 先要有一台服务器
- 再安装`docker`与`docker-compose`
- 再打开服务器防火墙的`9000`端口
- 再拉去一下本仓库
- 再按照示例自定义配置文件`./config/config.env`和`./docker-compose.yml`
- 再运行
- 再进入控制面板`http://你的ip:9000`
- 再登录你的不常用小号
- 再根据提示完成功能向导
- 就差不多了
